# Portfolio Frontend

Modern React frontend with Tailwind CSS and Framer Motion animations.

## Features
- Modern, responsive design with Tailwind CSS
- Smooth animations with Framer Motion
- Mobile-first approach
- Contact form integration
- Sections for about, projects, skills, and experience
- Glass morphism UI components
- Fluid, interactive controls

## Setup

### 1. Install Dependencies
```bash
cd frontend
npm install
```

### 2. Create Environment File
```bash
cp .env.example .env.local
```

### 3. Run Development Server
```bash
npm run dev
```

Access the site at `http://localhost:3000`

### 4. Build for Production
```bash
npm run build
```

## Pages

- **Home** - Hero section with featured projects
- **About** - Skills, experience, and education
- **Projects** - Portfolio projects with filtering
- **Contact** - Contact form with message submission

## Components

- **Header** - Navigation with smooth animations
- **Footer** - Social links and copyright
- **ProjectCard** - Project showcase with hover effects
- **SkillBar** - Animated skill proficiency indicator
- **ExperienceCard** - Work experience display
- **EducationCard** - Education information

## Styling

- Tailwind CSS for utility-first styling
- Custom animations and glass morphism effects
- Responsive design for all screen sizes
- Dark theme with cyan accents
